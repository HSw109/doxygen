\doxysubsection{Detailed Description}
Represents an event in the network simulation. 

This class models an event, such as packet arrival or transmission, with details about the destination and time.\hypertarget{classevent_autotoc_md1}{}\doxysubsubsubsubsubsection{\texorpdfstring{Attributes\+:}{Attributes\+:}}\label{classevent_autotoc_md1}

\begin{DoxyItemize}
\item int m\+\_\+type\+: ~\newline
 The type of the event\+: 0 for arrival, 1 for transmission.
\item destination m\+\_\+dest\+: ~\newline
 For an arrival event, this is the corresponding destination.
\item double m\+\_\+time\+: ~\newline
 The time when the event occurs. ~\newline
 
\end{DoxyItemize}
