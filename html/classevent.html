Detailed Description

Represents an event in the network simulation.

This class models an event, such as packet arrival or transmission, with details about the destination and time.

Attributes:
  • int m_type:
    The type of the event: 0 for arrival, 1 for transmission.
  • destination m_dest:
    For an arrival event, this is the corresponding destination.
  • double m_time:
    The time when the event occurs.
